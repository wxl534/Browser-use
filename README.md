# Browser-use
# 分析原有代码各个文件夹功能
# browser_use/
## actor/ - 浏览器操作执行器（元素操作、鼠标、页面控制）
## agent/ - AI Agent 核心逻辑（消息管理、提示词、服务层）
## browser/ - 浏览器管理（会话管理、云浏览器、监控）
## code_use/ - 代码使用和执行功能
## controller/ - 控制器接口
## dom/ - DOM 树处理和序列化
## filesystem/ - 文件系统操作
## integrations/ - 第三方集成（如 Gmail）
## llm/ - 各 LLM 提供商的适配器（OpenAI、Anthropic、Google 等）
## mcp/ - MCP (Model Context Protocol) 支持
## sandbox/ - 沙箱环境
## screenshots/ - 截图服务
## skill_cli/ - CLI 工具
## skills/ - 技能系统
## sync/ - 数据同步
## telemetry/ - 遥测和监控
## tokens/ - Token 管理和计费
## tools/ - 工具注册和使用
# docs/ 完整的用户文档
## customize/ - 自定义指南（Agent、Browser、Tools 等）
## development/ - 开发文档
## examples/ - 文档中的示例
## 各种 .mdx 文件：快速开始、介绍、生产部署等
